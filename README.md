# KeyMouseSimulation
Go 实现的鼠标键盘按键模拟器 |  Mouse and Keyboard simulator

## 界面
![Image](https://github.com/ljj1505204588/KeyMouseSimulation/raw/main/中文界面.png)

![Image](https://github.com/ljj1505204588/KeyMouseSimulation/raw/main/英文界面.png)

## 功能 
- 鼠标键盘录制回放
- 鼠标路径取消记录
- 速度调节
- 回放次数设置
- 热键修改
